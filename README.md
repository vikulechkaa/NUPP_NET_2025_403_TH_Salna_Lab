# Лабораторна робота №1 — Основи C# та ООП

## Тема
Бібліотека (Library) — модель предметної області та дженерік CRUD-сервіс.

## Мета роботи
Закріпити навички програмування на C# та застосування ООП:
- класи та наслідування
- інтерфейси
- колекції .NET
- дженерики
- події та делегати
- статичні поля/методи
- методи розширення

## Структура проєкту
- `Library.Common` — моделі та сервіс CRUD
- `Library.Console` — консольна демонстрація роботи CRUD

## Реалізовано в моделі (вимоги)
- Модель містить 4+ класи, є наслідування (мінімум 2 похідні класи)
- Є конструктори та методи у класах
- Є статичне поле та статичний конструктор
- Є делегат та подія
- Є статичні методи
- Є метод розширення

## CRUD-сервіс
Реалізовано `ICrudService<T>` та `InMemoryCrudService<T>`:
- Create / Read / ReadAll / Update / Remove
- (Додатково) Save/Load у файл (серіалізація)

## Як перевіряти
1. Відкрити код у папках `Library.Common` та `Library.Console`
2. Запустити консольний застосунок (за наявності .NET SDK)
3. Результат виконання наведено у файлі `Lab1_Result.pdf`
